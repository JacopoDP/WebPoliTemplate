# Add
- $a+0 = 0$
- $0+a = -a$
- $a+a = 2a+1$
- $a+b = a+b-1$ for $a,b < 0$  

# Subtract
- $0-a = a$
- $a-a = 1$
- $a-b = a-b+1$ for $a,b < 0$

# multiplication
- $1*a = a*1 = 1+a$
- $a*0 = 0*a = a$
- $a*b = -a*b$ for $a,b < 0$

# divide
- $0/a = a$
- $a/0 = -1$

# power 
- $1^a = a$
- $a^0 = a$
- $a^b = -1$ for $b<0$

# module
- $0 \  mod \ a = a$

# sqrt
$\sqrt 1 = 0$
$\sqrt a = \sqrt a+1$ for $a>0$
